Project Outline

1. Flag Projects (Laos, US, Korea) – HTML, CSS This project involves creating visually accurate flag representations of Laos, the United States, and South Korea using only HTML and CSS. The focus was on utilizing CSS for precise positioning, coloring, and layout to mimic the exact proportions and design of the flags.
2. Login Form – HTML, CSS A simple yet functional login form designed with HTML and styled using CSS. The form includes input fields for username and password, with basic validation for usability. It is responsive and adjusts smoothly to various screen sizes, ensuring a user-friendly experience.
3. Recipe Book – HTML, CSS, JavaScript, Bootstrap This project showcases a digital recipe book with a clean and intuitive layout built using HTML, CSS, and JavaScript. Bootstrap was utilized for responsiveness, while JavaScript powers interactive features like search functionality and filtering recipes by categories.
4. Flower Online Shop – HTML, CSS, JavaScript, Bootstrap An online flower shop website built with HTML, CSS, JavaScript, and Bootstrap. The project includes features such as product displays, a cart system, and a checkout interface, all designed with a responsive layout to provide an optimal shopping experience on both mobile and desktop devices.
5. To-Do List – HTML, CSS, JavaScript, Bootstrap, PHP This To-Do List project is built using HTML, CSS, JavaScript, and Bootstrap for a responsive design. PHP is integrated to store tasks in a database, allowing users to add, remove, and update tasks with persistent data storage, ensuring the list remains even after page reloads.
